Privacy Policy

Last updated: April 8, 2026

1. Overview

UI Prompt Picker is a browser extension that runs locally in your browser. It helps users browse UI examples, generate prompts, copy code, and optionally send prompts to third-party AI services.

This extension does not operate a backend server for the purpose of collecting user data.

2. Information Collection and Storage

This extension does NOT collect, transmit, or store personal data on any developer-controlled servers.

All data handled by the extension remains within the user's browser and is stored locally using browser extension storage APIs.

The extension may store:
- UI preferences (e.g. theme, framework selection)
- Favorites or saved items
- Current category or interface state

This data is never automatically transmitted outside the user's device.

3. Permissions Usage

The extension requests the following permissions:

- storage  
  Used to save user preferences and favorites locally.

- tabs  
  Used to open the extension interface and to open third-party AI websites when the user explicitly chooses to do so.

4. Clipboard Access

The extension writes to the clipboard only when the user clicks a “copy” button.

It does not read clipboard data and does not access the clipboard without user interaction.

5. Interaction with Third-Party Services

The extension provides an optional feature that allows users to send generated prompts to third-party AI platforms such as ChatGPT, Claude, Gemini, or Grok.

When the user uses this feature:
- The generated prompt is included in the URL and sent to the selected service
- The data is handled by that third-party service, not by this extension
- The developer does not receive or store this data

Note: Prompts sent via URL may be visible in your browser history, network logs, or intermediary systems (such as proxies).

Users should review the privacy policies of those services if they choose to use this feature.

6. What We Do NOT Do

This extension does not:
- Collect personally identifiable information
- Track browsing history
- Access cookies or authentication data
- Monitor user activity in the background
- Inject advertisements or tracking scripts
- Send analytics or telemetry data to any server

7. Children's Privacy

This extension is not directed at children under the age of 13 and does not knowingly collect data from children.

8. Changes to This Policy

This Privacy Policy may be updated if the extension’s functionality changes. The latest version will be available on the extension’s store listing page.
